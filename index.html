PLUS Sorteos

Comisión: 30345

Alumno: Gabriel Eduardo Cañadas Fragapane

Esta es una herramienta online donde podrás obtener ganadores aleatorios (al azar) de una lista de participantes.

Para el proyecto final se realizará una página que permita realizar sorteos a partir de una cierta cantidad de nombres o números que ingrese el usuario.

  • Se deberá indicar cuantos ganadores se desean (máximo 5)
  • Se debera indicar si se desea sortear números o nombres
  • Se deberán ingresar todos los números o nombres a participar del sorteo
  • Se mostrarán todos los ganadores
